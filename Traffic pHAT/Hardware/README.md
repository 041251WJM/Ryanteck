# RTK Traffic pHAT

## Traffic pHAT Pinout

- Red LED - GPIO25 (pin 22)
- Yellow LED - GPIO24 (pin 18)
- Green LED - GPIO23 (pin 16)

## GPIO Pins Used

## EEPROM
The Traffic pHAT contains NO EEPROM, this was so it doesn't cause any conflicts with other HATs or pHATs.
